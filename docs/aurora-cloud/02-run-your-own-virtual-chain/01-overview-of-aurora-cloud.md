# Overview of Aurora Cloud

[Aurora Cloud](https://auroracloud.dev/) is the platform facilitating the management of your Virtual Chains.

:::info
_To learn more about Aurora, start with_ [_this introductory article._](/aurora-cloud/welcome/introduction)
:::

On Aurora Cloud you can monitor your chain, access its parameters, explore, enable and configure add-ons, all from one simple interface.

![frame827_2](/img/.gitbook/assets/Frame_827_(2).png)

There are 8 major components to Aurora Cloud, all supported by default by all virtual chains:

1. [**Chain customisation**](/aurora-cloud/run-your-own-virtual-chain/customise-your-chain/)
2. [**Permissions management**](/aurora-cloud/run-your-own-virtual-chain/manage-permissions/)
3. [**Gas abstraction**](/aurora-cloud/run-your-own-virtual-chain/gas-abstraction)
4. [**On/offramp**](/aurora-cloud/run-your-own-virtual-chain/on-offramp-to-your-chain)
5. [**Oracle**](/aurora-cloud/run-your-own-virtual-chain/pyth-oracle)
6. [**Integration with Centralised Exchanges**](/aurora-cloud/run-your-own-virtual-chain/connect-cexes-to-your-chain)
7. [**Bridge support**](/aurora-cloud/run-your-own-virtual-chain/bridge-assets-to-your-chain)
8. [**Access to global liquidity through NEAR Intents**](https://docs.near-intents.org/near-intents/)
